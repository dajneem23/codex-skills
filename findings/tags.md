# Tags (Controlled Vocabulary)

Primary categories (use one):
- `auth` — access control, role binding, ownership checks.
- `accounting` — balance sheet, invariants, fee/interest, ERC4626 share math.
- `oracle` — price feeds, freshness/confidence, TWAP/spot manipulation.
- `replay` — signatures, nonce lifecycle, domain binding, front-run/replay.
- `reentrancy` — CEI violations, callbacks, cross-function/state sharing.
- `dos` — griefing, unbounded loops, pause/edge-state blocking.
- `upgrade` — proxy/UUPS/beacon issues, initialization, storage layout.
